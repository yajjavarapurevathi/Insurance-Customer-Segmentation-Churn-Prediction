# Insurance Customer Segmentation & Churn Prediction

## Project Overview
This project analyzes insurance customer behavior to identify meaningful customer groups, understand claim patterns, and predict churn risk. The goal is to help insurance companies improve customer retention, manage risk, and make data-driven business decisions.

## Business Problem
Insurance companies manage many policyholders with different premium amounts, claim histories, payment behaviors, satisfaction levels, and risk profiles. Treating every customer the same can lead to missed retention opportunities and poor risk management.

This project helps answer:
- Which customers are high-value and loyal?
- Which customers are likely to churn?
- Which customers have high claim or payment risk?
- Which customer groups should receive retention offers?

## Project Goals
- Create a realistic insurance customer dataset
- Perform data cleaning and exploratory data analysis
- Segment customers using K-Means clustering
- Use PCA to visualize customer segments
- Build a churn prediction model
- Generate KPI reports for business insights
